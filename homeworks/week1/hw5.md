## 請解釋後端與前端的差異。
1. 前端就是網頁我們看得到的部分都叫前端，頁面的顏色、設計等。前端工程師會與 UI 工程師合作，由 UI 工程師說明網頁欲呈現的內容，再由前端工程師透過 JavaScript、HTML、CSS 做網頁來。
2. 後段就是泛指網頁看不到的部分，例如資料庫的設定、處理好 API 等工作，這會與商業邏輯比較相關。
